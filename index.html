Overall Architecture

Secure Data Exchange through Hospitals

The diagram shows an overall architecture of transferring encrypted data with the Tensorflow and SCONE platforms.

The data from the database is encrypted using FSPF(FILE SYSTEM PROTECTION FILE) and then uploaded into the cloud. The Tensorflow and SCONE platform ensures the security and confidentiality of the data, which is stored in the enclave. The attackers with root access cannot access the data. As shown in the architecture, several hospitals could exchange data in a secure way.
